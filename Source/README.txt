# Yêu cầu môi trường: Python >= 3.8.0 và PyTorch >= 1.8

# Tải và cài đặt YOLOv5:
git clone https://github.com/ultralytics/yolov5
cd yolov5
pip install -r requirements.txt

# Cách sử dụng chương trình:
1. Nhận diện biển số: python main.py --img [img_path]
với [img_path] là đường dẫn của ảnh đầu vào.
